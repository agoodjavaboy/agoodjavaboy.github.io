安装与渲染画布 ​

本次demo将使用 JS、vue3、vite、pnpm 来搭建基础的框架。

初始化项目 ​

  1. 使用vite进行项目初始化：

    sh
    pnpm create vite # 开始初始化项目指令
√ Project name: ... pixi-vue3-vite # 项目的名称
√ Select a framework: » Vue # 基础框架使用Vue
√ Select a variant: » JavaScript # 使用JS作为开发语言

  2. 根据指示进行依赖的安装：

    sh
    cd pixi-vue3-vite # 从当前目录进入到刚刚创建的项目的目录中
pnpm install # 进行依赖的安装
pnpm run dev # 启动项目

  3. 向项目中安装 pixi 并引入到页面：

    sh
    pnpm i --save pixi.js

    此时，package.js 文件应该类似下面的内容：

    json
    {
  "name": "pixi-vue3-vite",
  "private": true,
  "version": "0.0.0",
  "type": "module",
  "scripts": {
    "dev": "vite",
    "build": "vite build",
    "preview": "vite preview"
  },
  "dependencies": {
    "pixi.js": "^7.4.0",
    "vue": "^3.4.19"
  },
  "devDependencies": {
    "@vitejs/plugin-vue": "^5.0.4",
    "vite": "^5.1.4"
  }
}

渲染画布 ​

如果正确的话，vite-vue会默认给予一个展示模板，我们只需要在 App.vue 上编辑代码就可以放内容到网页上。

所以将 App.vue 改装成以下内容：

html
<script setup>
  import * as PIXI from 'pixi.js';//导入pixi

  //创建画布
  const app = new PIXI.Application({
    background: '#1099bb',//画布的背景色
    resizeTo: window,//随界面大小自动调整
  });

  //将画布放到整个浏览器
  document.body.appendChild(app.view);
</script>

<template>
  
</template>

<style>
  *{
    margin: 0;
    overflow: hidden;
  }
</style>

顺便把模板给的样式也关闭，这些样式在 main.js 中，删除即可：

js
import { createApp } from 'vue'
import './style.css' //删除样式
import App from './App.vue'

createApp(App).mount('#app')

当然还可以把模板附送的 HelloWord.vue 文件删掉，这并不是必要的。

接着就可以重启项目，就可以看到一个 #1099bb 色的网页了，这将是未来我们创作的画布。
