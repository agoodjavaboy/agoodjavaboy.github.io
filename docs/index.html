вҖҚрҹ’»ж–ҮжЎЈдёӯеҝғрҹ•өпёҸвҖҚ

рҹ‘“еҗ‘з§ҜжһҒжӢјжҗҸзҡ„дәәз”ҹиҮҙж•¬рҹҗұвҖҚ

е°Ҹз”өи§Ҷйў‘йҒ“ рҹҘ° (opens new window)

рҹ‘Җз«Ӣиә«д№Ӣжң¬

жғіеӯҰзҡ„дёңиҘҝзү№еҲ«еӨҡеҸҲжҮ’еҫ—еҺ»еӯҰпјҢеҲ°еӨҙжқҘд»Җд№ҲйғҪдёҚжҳҜгҖӮж—¶й—ҙеӨҡзҡ„жҳҜпјҢжӢҝиө·жқҘејҖе§ӢеӯҰе•ҠпјҢе№ҙиҪ»дёҚжӢјеҚҡпјҢзңӢеҲ°жңәдјҡдёҚдәүеҸ–пјҢдәәжІЎжңүжўҰжғіе°ұжҳҜдёҖжқЎе’ёйұјгҖӮеҸҜиғҪдҪ дёҚжҳҜд№ жғҜе®үйҖёпјҢе°ұжҳҜи®ЁеҺҢеӯҰд№ иҖҢе·ІгҖӮ

рҹ‘№еҒңжӯўж— дёә

еҝ«иҠӮеҘҸзҹӯи§Ҷйў‘е’ҢдҝЎжҒҜеҢ–е№іеҸ°еңЁж¶ҲзЈЁдҪ зҡ„ж„Ҹеҝ—еҠӣе’ҢиҖҗеҠӣпјҢеңЁз»ҷйҷҢз”ҹдәәеёҰжқҘи®ҝй—®йҮҸд№ӢеҗҺдҪ еҸӘиғҪеҪ“дёӘеәҹжҹҙпјҢз”ҡиҮійғҪеҫ—дёҚеҲ°еҝ«д№җгҖӮжңүдәӣдәӢй•ҝж•ҲдҪҶз—ӣиӢҰпјҢиҖҢеҸҰдёҖдәӣдәӢеҫҲзҹӯж—¶й—ҙиғҪиҺ·еҫ—еҝ«д№җпјҢдҪҶжөӘиҙ№зҡ„ж—¶й—ҙиҝҳиҰҒиҮӘе·ұеҒҝиҝҳгҖӮ

рҹ“ҡйҳ…иҜ»ж°ёж— жӯўеўғ

еҸҜд»ҘжІЎжңүж–ҮеҮӯпјҢдҪҶдёҚиғҪжІЎжңүж–ҮеҢ–пјҢж— и®әеңЁдё“дёҡйўҶеҹҹиҝҳжҳҜж—ҘеёёйғҪи·ҹдәәиҒҠдёҚеҲ°дёҖеқ—еҺ»пјҢж—¶й—ҙйғҪжөӘиҙ№еҲ°е“ӘйҮҢеҺ»дәҶпјҢжғізңӢзҡ„д№ҰзңӢдәҶд№ҲпјҢжғіиҖғзҡ„иҜҒиҖғдәҶд№ҲпјҢеҚ•иҜҚзҹҘиҜҶзӮ№йғҪиғҢдәҶеҗ—пјҢе…¶е®һиҝҷдәӣйғҪдёҚз”ЁжҲ‘еҺ»жҸҗйҶ’зҡ„гҖӮ

рҹ‘ҫжүҫеҲ°ж–°дё–з•Ңзҡ„й’ҘеҢҷ

дҪ иҜҙдҪ жҳҜж–°йқ’е№ҙпјҢеҚҙдёҚиһҚе…Ҙж•ҙдёӘдё–з•ҢгҖӮжҳҜдёҚиһҚе…Ҙеҗ—пјҹеҸҜиғҪжҳҜеӣ дёәеӨ–иҜӯзҡ„йҡҫеӯҰиҖҢжңӣиҖҢеҚҙжӯҘпјҢжңҖз»ҲиҝҳжҳҜзңјзңӢзқҖиў«еҲ«дәәиө¶и¶…д№ҹеқҡжҢҒиәәе№ігҖӮзҡ„зЎ®жҜҚиҜӯе·Із»Ҹж №ж·ұи’ӮеӣәпјҢе№¶дё”еӨ–иҜӯдҪңдёәдёҖй—ЁиҜӯиЁҖиӮҜе®ҡжңүз№Ғзҗҗзҡ„зҹҘиҜҶзӮ№пјҢдҪҶдёҖеҲҮйғҪж— жі•йҒҝе…ҚгҖӮ

рҹ‘ҪиҮӘеҫӢж— йңҖйӮЈд№ҲйҡҶйҮҚ

еҸҜиғҪеңЁжһҒе°Ҹзҡ„дәӢжғ…дёҠжҲ‘д»¬йғҪж— жі•еҒҡеҲ°иҮӘеҫӢпјҢжӣҙдёҚиҰҒиҜҙеӯҰд№ гҖҒй”»зӮјгҖҒеқҡжҢҒдёҖдёӘзҲұеҘҪпјӣжҢүж—¶еҗғйҘӯжҢүж—¶зқЎи§үд№ҹжҳҜиҮӘеҫӢпјҢжҢүж—¶еҒҡиҜҘеҒҡзҡ„е°ұжҳҜжңҖеҘҪзҡ„иҮӘеҫӢпјҢд№ҹж— йңҖжңүеӨҡеӨ§жҲҗе°ұпјҢе…ҲиҰҒжңүи§„еҫӢгҖӮ

рҹҷҠд»Җд№ҲеҸ«жңүиғҪеҠӣ

зңҹжӯЈзҡ„жңүиғҪеҠӣ=жҠҖжңҜејә+дәәи„үе№ҝпјҢе…үиҜҙдёҚз»ғе’Ңе…үз»ғдёҚиҜҙжІЎжңүдёҖдёӘжҳҜиӨ’д№үиҜҚпјҢз”ҹжҙ»дёҚд»…д»…жҳҜз®ҖеҚ•зҡ„еӯҰд№ е’Ңй—·еӨҙе·ҘдҪңпјҢиҰҒеӯҰдјҡеҖҫеҗҗе’ҢжІҹйҖҡпјҢеҗҢж ·иҰҒеӯҰдјҡеҝҚиҖҗе’Ңдҝқз•ҷгҖӮ
