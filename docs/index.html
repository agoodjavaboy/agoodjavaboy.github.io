👀立身之本

想学的东西特别多又懒得去学，到头来什么都不是。时间多的是，拿起来开始学啊，年轻不拼博，看到机会不争取，人没有梦想就是一条咸鱼。可能你不是习惯安逸，就是讨厌学习而已。

👹停止无为

快节奏短视频和信息化平台在消磨你的意志力和耐力，在给陌生人带来访问量之后你只能当个废柴，甚至都得不到快乐。有些事长效但痛苦，而另一些事很短时间能获得快乐，但浪费的时间还要自己偿还。

📚阅读永无止境

可以没有文凭，但不能没有文化，无论在专业领域还是日常都跟人聊不到一块去，时间都浪费到哪里去了，想看的书看了么，想考的证考了么，单词知识点都背了吗，其实这些都不用我去提醒的。

👾找到新世界的钥匙

你说你是新青年，却不融入整个世界。是不融入吗？可能是因为外语的难学而望而却步，最终还是眼看着被别人赶超也坚持躺平。的确母语已经根深蒂固，并且外语作为一门语言肯定有繁琐的知识点，但一切都无法避免。

👽自律无需那么隆重

可能在极小的事情上我们都无法做到自律，更不要说学习、锻炼、坚持一个爱好；按时吃饭按时睡觉也是自律，按时做该做的就是最好的自律，也无需有多大成就，先要有规律。

🙊什么叫有能力

真正的有能力=技术强+人脉广，光说不练和光练不说没有一个是褒义词，生活不仅仅是简单的学习和闷头工作，要学会倾吐和沟通，同样要学会忍耐和保留。
